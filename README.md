## قائمة الدول (ISO 3166)

قائمة دول العالم باللغة العربية ، القائمة تشمل:
* الاسم بلغة الدولة المحلية
* الاسم باللغة العربية
* الاسم باللغة الإنجليزية
* مفتاح الإتصال
* رمز الدولة الثنائي حسب تصنيف ISO 3166


القائمة متوفرة بالصيغ التالية:
* sql
* xml
* json
* csv
* html
